## Description
### Subject
Landscape generation using Matlab.

### Features
* Perspective
* Painter algorithm :
	- result rendered using triangles
	- Phong light algorithm
	- No sunlight
* zBuffer algorithm :
	- result rendered using triangles
	- Phong light algorithm
	- Sunlight
	- Some black dots due to some bugs with the given API
